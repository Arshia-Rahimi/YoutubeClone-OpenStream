# Openstream

An open source andorid front-end for youtube.

## Screenshots


## features

- ad-free and backgound playback
- audio only playback
- view and save online playlists locally
- create and edit local playlist
- view watch history
- subscribe to channels locally

## libraries

- [NewPipeExtractor](https://github.com/TeamNewPipe/NewPipeExtractor) for scraping youtube
- media3 for video and audio playback
- Jetpack compose for ui
- Room for persistent storage
- DataStore for preferences
- Koin for dependency injection
- ktor http client
- coil for image caching
